# MidSoul-Wiki (午夜灵魂 维基)

![Logo](./static/img/MidSoul-Logo.png)

MidSoul-Wiki 是 [HeartFireProject](https://github.com/Heart-Fire-Project) 旗下项目 **《午夜灵魂》(MidSoul)** 的官方文档与维基站点。

本项目使用 [Docusaurus 3](https://docusaurus.io/) 构建，旨在为玩家和开发者提供关于《午夜灵魂》机制、能力、进度及饰品集册的详尽参考资料。

## 📖 核心板块

- **整体介绍**: 了解《午夜灵魂》的世界观与核心概念。
- **机制说明**: 深入理解游戏的核心运行逻辑。
- **能力一览**: 全面的角色/生物能力索引。
- **饰品集册**: 详细的饰品属性与获取方式。
- **进度碑刻**: 追踪开发进度与重要里程碑。
- **回响记录**: 归档重要的社区反馈与版本变动。

## 🛠️ 本地开发

确保你已安装 [Node.js](https://nodejs.org/) (建议 v20 或更高版本)。

1. **克隆仓库**
   ```bash
   git clone https://github.com/HeimNad/MidSoul-Wiki.git
   cd MidSoul-Wiki
   ```

2. **安装依赖** (本项目使用 pnpm)
   ```bash
   pnpm install
   ```

3. **启动开发服务器**
   ```bash
   pnpm start
   ```

4. **构建静态站点**
   ```bash
   pnpm build
   ```
